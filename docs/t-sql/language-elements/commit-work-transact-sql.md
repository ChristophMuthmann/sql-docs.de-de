# <a name="commit-work-transact-sql"></a>COMMIT WORK (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-asdb-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-asdb-xxxx-xxx-md.md)]

  Markiert das Ende einer Transaktion.  
  
 ![Themenlinksymbol](../../database-engine/configure-windows/media/topic-link.gif "Topic link icon") [Transact-SQL Syntax Conventions (Transact-SQL-Syntaxkonventionen)](../../t-sql/language-elements/transact-sql-syntax-conventions-transact-sql.md)  
  
## <a name="syntax"></a>Syntax  
  
```  
  
COMMIT [ WORK ]  
[ ; ]  
```  
  
## <a name="remarks"></a>Remarks  
 Diese Anweisung funktioniert genau wie COMMIT TRANSACTION, mit dem Unterschied, dass COMMIT TRANSACTION einen benutzerdefinierten Transaktionsnamen akzeptiert. Diese COMMIT-Syntax, mit oder ohne Angabe des optionalen Schl√ºsselworts WORK, ist mit SQL-92 kompatibel.
