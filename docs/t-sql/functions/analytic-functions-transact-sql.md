# <a name="analytic-functions-transact-sql"></a>Analytische Funktionen (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2012-all-md](../../includes/tsql-appliesto-ss2012-all-md.md)]

SQL Server unterstützt die folgenden analytischen Funktionen. Analytische Funktionen berechnen auf Grundlage einer Gruppe von Zeilen einen Aggregatwert. Sie können jedoch im Gegensatz zu Aggregatfunktionen mehrere Zeilen für jede Gruppe zurückgeben. Sie können analytische Funktionen verwenden, um gleitende Durchschnitte, laufende Summen, Prozentsätze oder die ersten N-Ergebnisse innerhalb einer Gruppe zu berechnen.
