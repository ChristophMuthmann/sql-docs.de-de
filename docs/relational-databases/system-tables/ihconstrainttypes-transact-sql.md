# <a name="ihconstrainttypes-transact-sql"></a>IHconstrainttypes (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Die **IHconstrainttypes** -Systemtabelle enthält eine Zeile für jeden Typ von nicht - SQL Server-Einschränkung für nicht - SQL Server-Verleger unterstützt. Diese Tabelle wird in der Verteilungsdatenbank gespeichert.  
  
|Spaltenname|Datentyp|Description|  
|-----------------|---------------|-----------------|  
|**type**|**nvarchar(255)**|Name des unterstützten Typs von Nicht-SQL Server-Einschränkung|
