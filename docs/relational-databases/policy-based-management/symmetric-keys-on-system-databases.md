# <a name="symmetric-keys-on-system-databases"></a>Symmetrische Schlüssel für Systemdatenbanken
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  Diese Regel überprüft vom Benutzer erstellte symmetrische Schlüssel in den Datenbanken master, msdb, model und tempdb.  
  
## <a name="best-practices-recommendations"></a>Empfehlungen zu Best Practices  
 Erstellen Sie keine symmetrischen Schlüssel in den Systemdatenbanken.
