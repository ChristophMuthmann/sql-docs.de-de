# <a name="serversettingsgeneralflag-class"></a>ServerSettingsGeneralFlag-Klasse
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Die [ServerSettingsGeneralFlag-Klasse](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) stellt die allgemeinen Flageinstellungen in einer Instanz von [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Mit dieser Klasse können Sie folgende Vorgänge ausführen:  
  
-   Anzeigen des Flagnamens.  
  
-   Anzeigen und Ändern des Flagwerts.
