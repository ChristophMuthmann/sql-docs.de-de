# <a name="setint-method-sqlserverpreparedstatement"></a>SetInt-Methode (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Legt den angegebenen Parameter auf den angegebenen **Int** Wert.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public final void setInt(int n,  
                         int value)  
```  
  
#### <a name="parameters"></a>Parameter  
 *n*  
  
 Ein **Int** , der die Parameteranzahl angibt.  
  
 *value*  
  
 Ein **Int** Wert.  
  
## <a name="exceptions"></a>Ausnahmen  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Hinweise  
 Diese SetInt-Methode wird von der SetInt-Methode in der java.sql.PreparedStatement-Schnittstelle angegeben.
