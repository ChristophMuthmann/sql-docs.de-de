# <a name="getbinarystream-method-sqlserverresultset"></a>GetBinaryStream-Methode (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Ruft den Wert der angegebenen Spalte in der aktuellen Zeile dieses [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) Objekts als Binärdatenstrom nicht interpretierter Bytes.
