# <a name="first-method-sqlserverresultset"></a>Bei der ersten Methode (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Verschiebt den Cursor in die erste Zeile dieser [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) Objekt.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public boolean first()  
```  
  
## <a name="return-value"></a>RÃ¼ckgabewert  
 **"true"** , wenn der Cursor in die erste Zeile bewegt wird. Andernfalls lautet der Wert **false**.

 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Hinweise  
 Diese erste Methode wird von der ersten Methode in der java.sql.ResultSet-Schnittstelle angegeben.
