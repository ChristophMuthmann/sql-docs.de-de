# <a name="concurssscrolllocks-field-sqlserverresultset"></a>CONCUR_SS_SCROLL_LOCKS-Feld (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Zur Angabe einer [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] Lese-/Schreibzugriff Typs der vollständigen Parallelität mit Sperren auf Zeilenebene.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public static final int CONCUR_SS_SCROLL_LOCKS  
```  
  
## <a name="field-value"></a>Feldwert  
 Ein **Int** Wert von 1009.
