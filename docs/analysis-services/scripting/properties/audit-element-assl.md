# <a name="audit-element-assl"></a>Audit-Element (ASSL)
[!INCLUDE[ssas-appliesto-sqlas](../../../includes/ssas-appliesto-sqlas.md)]
  Gibt an, dass ein [Trace](../../../analysis-services/scripting/objects/trace-element-assl.md) -Element selbst dann keine Ereignisse löschen kann, wenn dies eine verminderte Leistung auf dem Server verhindern könnte.  
  
## <a name="syntax"></a>Syntax  
  
```xml  
  
<Trace>  
   ...  
   <Audit>...</Audit>  
   ...  
</Trace>  
```  
  
## <a name="element-characteristics"></a>Elementmerkmale  
  
|Merkmal|Beschreibung|  
|--------------------|-----------------|  
|Datentyp und -länge|Boolean|  
|Standardwert|**False**|  
|Kardinalität|0-1: Optionales Element, das nur einmal auftreten kann.|

|Beziehung|Element|  
|------------------|-------------|  
|Übergeordnetes Element|[Trace](../../../analysis-services/scripting/objects/trace-element-assl.md)|  
|Untergeordnete Elemente|Keine|
