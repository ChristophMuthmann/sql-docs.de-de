|Konstante|Wert|Description|  
|--------------|-----------|-----------------|

|**adRINone**|0|Standard. Es wird keine Aktion ausgeführt.|

|**adRISetNull**|2|Foreign Key-Wert wird festgelegt, auf Null.|
