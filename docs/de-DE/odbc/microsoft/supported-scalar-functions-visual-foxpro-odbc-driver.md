Der Visual FoxPro-ODBC-Treiber unterstützt drei Typen von Skalarfunktionen, gemäß der Definition in der ODBC-SQL-Grammatik: Zeichenfolgen-, numerischen und Uhrzeit- und Datumsfunktionen.
