# <a name="lesson-1-basic-navigation-in-database-engine-tuning-advisor"></a>Lektion 1: Grundlagen zur Navigation im Datenbankoptimierungsratgeber
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Im Datenbankoptimierungsratgeber werden Optimierungssitzungen und Berichte zu Optimierungsempfehlungen über eine grafische Benutzeroberfläche angezeigt. In dieser Lektion erfahren Sie, wie Sie das Tool starten und die Anzeige konfigurieren. Am Ende der Lektion kennen Sie verschiedene Möglichkeiten zum Starten des Tools. Sie wissen, wie Sie die Anzeige konfigurieren, sodass die Optimierungsaufgaben unterstützt werden, die Sie regelmäßig ausführen.  
  
Diese Lektion enthält die folgenden Themen:  
  
-   Starten des Datenbankoptimierungsratgebers  
  
-   Festlegen von Optionen und Layout
